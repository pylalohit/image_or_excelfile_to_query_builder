 2.2 All the attribute names of the respective columns should be in the first row.
 2.3 All the data types of the respective columns should be in the last row.

 3.1 Image should be clear and crisp without any distortion.

 3.3 After clicking download the excel file will be downloaded to your pc.
 3.4 Review all the cells in the output file and check there are no errors and edit the desired changes and save the excel file and then upload the excel file to Query Builder or follow steps 2 to get the queries as output.
